# 智能销售数据分析系统 - 技术报告

## 1. 项目概述

### 1.1 项目背景

随着大数据时代的到来，企业对数据分析的需求日益增长。传统的数据分析方式需要专业的技术人员编写复杂的SQL查询和代码，对普通用户不够友好。本项目旨在开发一个基于大语言模型的智能数据分析系统，让用户能够通过自然语言与数据进行交互，降低数据分析的门槛。

### 1.2 项目目标

- 实现一个完整的智能销售数据分析系统
- 提供自然语言问答、数据可视化、SQL查询等功能
- 使用大语言模型实现智能交互
- 构建用户友好的Web界面

### 1.3 技术选型

| 技术 | 选型 | 理由 |
|------|------|------|
| 后端框架 | Flask | 轻量级、易于扩展 |
| 前端 | HTML/CSS/JS | 快速开发、兼容性好 |
| 数据库 | SQLite | 轻量级、无需额外配置 |
| LLM | OpenAI API | 功能强大、兼容性好 |
| 可视化 | Matplotlib + Plotly | 静态+交互式图表 |
| 数据处理 | Pandas | 功能丰富、性能优秀 |

## 3. 核心功能实现

### 3.1 智能问答

**实现原理**：
1. 用户输入自然语言问题
2. 系统构建包含数据上下文的提示词
3. 调用LLM API获取回答
4. 返回结果并更新对话历史

**代码示例**：
```python
def chat(self, user_message: str, data_context: Optional[str] = None) -> str:
    messages = self._build_messages(user_message, data_context)
    response = self.client.chat.completions.create(
        model=self.model,
        messages=messages,
        temperature=0.7,
        max_tokens=2000
    )
    return response.choices[0].message.content
```

### 3.2 数据可视化

**实现原理**：
1. 前端发送图表类型和参数
2. 后端准备数据并生成图表
3. 保存图表文件并返回路径
4. 前端展示图表

**支持图表类型**：
- 柱状图：展示类别对比
- 折线图：展示趋势变化
- 饼图：展示占比分布

### 3.3 自然语言查询

**实现原理**：
1. 用户输入自然语言查询
2. LLM将查询转换为SQL语句
3. 执行SQL查询并返回结果
4. 展示查询结果和SQL语句

**示例**：
- 输入："查询所有电子产品的销售总额"
- 输出：`SELECT SUM(销售总额) FROM sales WHERE 产品类别 = '电子产品'`

### 3.4 数据概览

**展示指标**：
- 总销售额
- 订单数量
- 总利润
- 平均订单金额
